# Insanesec Website

Elite Security website with product catalog and checkout system.

## Features

- Product catalog with cart functionality
- Multiple payment options (PayPal, Bitcoin, Litecoin, Revolut)
- Responsive design with glassmorphism effects
- Social media integration (Discord, Telegram, YouTube, TikTok)

## Setup

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process required - pure HTML, CSS, and JavaScript

## Files

- `index.html` - Main homepage with products
- `checkout.html` - Checkout page with payment options
- `style.css` - Stylesheet (if exists)
- `favicon.jpg` - Website favicon

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Vanilla Tilt.js (for card animations)
